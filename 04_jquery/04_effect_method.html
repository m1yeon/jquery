04_제이쿼리_효과메서드(effect)

• 효과(effect) : 사용자가 지정한 html 요소를 선택하여 숨기거나 나타내기, 페이드 효과, 슬라이드 다운, 업, 애니메이션 기능을 사용하기 위한 메소드들이 있다

    • show(): : 선택한 요소를 보이게 한다.

    • hide(); - 선택한 요소를 숨긴다.

    • toggle(); - 선택한 요소를 숨기거나 나타낸다.

    • slideDown(); - 선택한 요소를 아래로 펼친다.

    • slideUp(); - 선택한 요소를 위로 접는다.

    • slideToggle(); - 선택한 요소를 아래로 펼치거나 위로 접는다.

    • fadeIn(); - 선택한 요소를 서서히 나타낸다.

    • fadeout(); - 선택한 요소를 서서히 사라지게한다.

    • fadeToggle(); - 선택한 요소를 서서히 나타내거나/사라지게한다.

    • fadeTo(); - 선택한 요소를 지정한 시간만큼 서서히 사라지게한다.
