02_제이쿼리 선택자(selector)

• 제이쿼리에서는 css에서 사용하던 선택자를 활용하여 서식을 적용 할 수 있는데, 제이쿼리는 css보다 좀 더 세밀하게 선택할 수 있고, 그 개수도 많음

• http://oscarotero.com/jquery/ : api참고사이트의 왼쪽 selectores에서 확인가능


2. 제이쿼리 선택자 연습하기

  • • 아이디 선택자(#id)
  • • 클래스 선택자(.class)
  • • 태그 선택자
  • • 그룹선택자
  • • 자손, 후손 선택자
  • • 구조선택자
  • • 동위선택자(+, ~)
  • • 탐색선택자(odd, even)
  • • 속성선택자(선택자[속성명=값], 선택자[속성])

3. css() 메소드 사용법

      // 서식이 1개일 때
      .css('속성','값');

      // 서식이 2개 이상일 때 1
      .css('속성','값').css('속성','값').css('속성','값');

      // 서식이 2개 이상일 때 2
        .css({'속성 : 값', '속성 : 값', '속성 : 값'});
